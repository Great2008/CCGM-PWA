# Font Files to Download

Go to https://gwfh.mranftl.com/fonts (Google Webfonts Helper)

## Lato
Search "Lato" → select subsets: **latin** → select styles: **300, regular, 700, 900**
Download the woff2 package and rename files:
- lato-v24-latin-300.woff2       → lato-300.woff2
- lato-v24-latin-regular.woff2   → lato-400.woff2
- lato-v24-latin-700.woff2       → lato-700.woff2
- lato-v24-latin-900.woff2       → lato-900.woff2

## Playfair Display
Search "Playfair Display" → select subsets: **latin** → select styles: **regular, italic, 700, 900**
Download the woff2 package and rename files:
- playfair-display-v37-latin-regular.woff2  → playfair-400.woff2
- playfair-display-v37-latin-italic.woff2   → playfair-400i.woff2
- playfair-display-v37-latin-700.woff2      → playfair-700.woff2
- playfair-display-v37-latin-900.woff2      → playfair-900.woff2

Place all 8 .woff2 files in /public/fonts/ alongside fonts.css
